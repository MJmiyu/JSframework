# JS Frameworks Course Assignment

Run with

```bash
npm start
```

Requires strapi to authenticate.\
`REACT_APP_STRAPI_LOGIN_URL` in `.env.development` sets url to default local strapi running.\
Also requires you have a user created in strapi, can use username or email as the username on login page.

Override `REACT_APP_STRAPI_LOGIN_URL` in `.env.production` for production setup.
